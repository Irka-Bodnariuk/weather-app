Міні-застосунок погоди.

Застосунок отримує дані з OpenWeatherMap API та відображає основну інформацію про погоду.

Технології:

1. React
2. Axios
3. CSS
4. OpenWeatherMap API

Функціонал:

1. Пошук погоди за назвою міста
2. Відображає:
   - назву міста,
   - іконку погоди,
   - температуру,
   - короткий опис погоди
   - швидкість вітру
3. Loader
4. Обробка помилки
